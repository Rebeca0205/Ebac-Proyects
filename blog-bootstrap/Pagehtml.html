HTML

El esqueleto de toda página web

HTML significa HyperText Markup Language o en español, Lenguaje de Marcado de Hipertexto. Es el lenguaje estándar para crear páginas web. Le dice al navegador cómo debe estructurarse y presentarse el contenido que vemos en Internet. No es un lenguaje de programación, como JavaScript o Python. Es un lenguaje de marcado, lo que significa que se usa para etiquetar elementos en una página, como párrafos, imágenes, títulos o enlaces.
